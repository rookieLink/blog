

具体业务场景： 
1、如何将下面的菜单层级铺平，再组装起来 (可在对象中添加任意属性)
```javascript
[
  {
    menu: '总菜单',
    childs: [
      {
        menu: '1',
        childs: [
          {
            menu: '1-1'
          },
          {
            menu: '1-2',
            childs: {
              menu: '1-2-1'
            }
          }
        ]
      },
      {
        menu: '2'
      }
    ]
  }
]
```

铺平示例：
```javascript
[
  { menu: '总菜单' },
  { menu: '1' },
  { menu: '1-1' },
  { menu: '1-2' },
  { menu: '1-2-1' },
  { menu: '2' }
]
```
