## 搭建框架ts类型

搭建ts类型框架命令
1、全局安装create-react-app环境
```md
npm install -g create-react-app
```

2、新建react-ts项目(现在react原生支持typecript，不再需要--scripts-version=react-scripts-ts这一套了)

```
create-react-app my-app --template typescript
```

3、将配置文件弹出到项目内部
```
npm run eject
```
