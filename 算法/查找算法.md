## 查找算法
### 1、二分查找法
时间复杂度为O(logn) ,这是一种及其高效的算法  
局限性： 
- 二分查找依赖的是顺序结构，简单点说就是数组，
- 其次，二分查找针对的是有序数组
- 再次，数据量太小不适合二分查找
- 最后数据量太大也不适合二分查找（因为要求连续）
``` c

public int bsearch(int[] a, int n, int value) {
  int low = 0;
  int high = n - 1;

  while (low <= high) {
    int mid = (low + high) / 2;
    if (a[mid] == value) {
      return mid;
    } else if (a[mid] < value) {
      low = mid + 1;
    } else {
      high = mid - 1;
    }
  }

  return -1;
}
```
